Mac Finder preferences

10 Aug 2015 dans Misc

Set prefered Layout

There are four different layouts that you can choose in the Finder.

  • Cover Flow Layout: Flwv
  • List View: Nlsv
  • Column View: clmv
  • Icon View: icnv

Where clmv is the layout name.
